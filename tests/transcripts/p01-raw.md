# HealthTrack Pitch Video Plan

> **Product:** AI-powered wearable for elderly fall detection & health monitoring
> **Company:** HealthTrack (Series A, $4M raised)
> **Audience:** Healthcare-focused VCs, 15-minute presentation slot
> **Ask:** $12M Series B → 50 facilities + DTC launch

---

## 1. Narrative Structure

### Story Arc

```
HOOK → PROBLEM → FAILED STATUS QUO → SOLUTION → HOW IT WORKS → PROOF → VISION → ASK
```

| Act | Beats | Emotional Register |
|-----|-------|--------------------|
| **Act I — The Stakes** (scenes 1–3) | Hook with human moment, establish the problem, show why current solutions fail | Tension, empathy |
| **Act II — The Answer** (scenes 4–6) | Introduce the product, show the tech, present traction data | Clarity, confidence |
| **Act III — The Future** (scenes 7–8) | Paint the vision, make the ask | Ambition, precision |

### Section Breakdown

| Scene | Title | Duration (est.) | Purpose |
|-------|-------|-----------------|---------|
| 01 | The Fall | ~60s | Emotional hook — a real scenario |
| 02 | The Numbers | ~60s | Scale the problem with data |
| 03 | The Broken Button | ~50s | Why current solutions fail |
| 04 | Meet HealthTrack | ~60s | Introduce the product |
| 05 | How It Works | ~70s | Technical differentiator |
| 06 | The Proof | ~90s | Traction, precision, NPS |
| 07 | The Vision | ~50s | 50 facilities, DTC, predictive health |
| 08 | The Ask | ~40s | $12M Series B, concrete next steps |
| | **Total** | **~8.5 min** | |

## 4. Visual Specifications

### Color Palette

| Token | Hex | Usage |
|-------|-----|-------|
| `--bg-dark` | `#0B1120` | Scenes 01, 02, 08 — emotional/bookend |
| `--bg-mid` | `#1C2333` | Scene 03 — clinical/problem |
| `--bg-light` | `#F5F5F5` | Scene 06 — data presentation |
| `--accent-teal` | `#0D7377` | Product, positive stats, CTA |
| `--accent-red` | `#E53935` | Negative stats, urgency |
| `--accent-green` | `#2E7D32` | Precision, success metrics |
| `--text-primary` | `#FFFFFF` | On dark backgrounds |
| `--text-dark` | `#212121` | On light backgrounds |

### Typography

| Role | Font | Weight | Size |
|------|------|--------|------|
| Large stats | Inter | 800 | 72px |
| Section headlines | Inter | 700 | 48px |
| Body text / cards | Inter | 400 | 28px |
| Small annotations | Inter | 400 | 20px |

### Element Types

| Type | Style |
|------|-------|
| **Stat card** | Rounded rect, slight shadow, large number + label below |
